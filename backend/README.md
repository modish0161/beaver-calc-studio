# BeaverCalc Studio Backend

Structural engineering calculations platform backend API.

## Features

- REST API for structural calculations
- JWT authentication with role-based access
- Plugin-based calculator system
- PostgreSQL database with SQLAlchemy
- Async processing with Redis + RQ

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

## API Documentation

Available at `/api/docs` when running.
